공짱은 한국 공공데이터 기반의 **엔터프라이즈 지오 데이터 플랫폼**.
2026-04-22 기준 신규 프로젝트, 디렉토리 뼈대 셋업 단계.

## 범위 (옵션 A 확정)

- 공공데이터를 가져와 **지도·표·차트·검색 UI로 보여주는 사이트**
- AI 자동 분석/보고서 생성 기능 **없음**
- 자연어 챗 인터페이스 **없음**
- 사용자 노출 텍스트는 데이터 그 자체 또는 사람이 작성한 것
- 상세: [scope_option_a.md](./scope_option_a.md)

## 핵심 데이터 도메인
- **필지 단위 공간 정보** (용도지역, 지구단위계획, 도시계획시설, 42개 법적지정)
- **법령·조례 정보** (국토계획법, 지자체 조례) — 검색·열람만, AI 해석 없음
- **건축물·토지 대장** (data.go.kr)
- **부동산 실거래가** (data.go.kr)

## 참조하는 선행 사례 (직접 카피 아님)
- korean-land-mcp, korean-law-mcp — 도메인 지식 reference로만 활용 (MIT)
- HiddenDeals44(X): AI 보고서 자동화 사례 — 우리는 옵션 A이므로 다른 길

**Why**: 사용자는 "하이엔드 엔터프라이즈 프로덕션 SSS급" 품질 요구. SSS급 핵심은 불필요한 복잡도(LLM 종속) 제거 + 감사·보안·컴플라이언스 명확성.
**How to apply**: 메인 시스템 기술 선택 시 LLM/AI/MCP 의존성 추가 거부. 데이터 fetch + 캐시 + UI 렌더링이 전부.
