# UI 라이팅 (한국어 해요체)

## 1. 톤

**해요체** 기본. 명령형 X, 권유형 O. 거리감 있는 존댓말 X.

| 상황 | ✅ OK | ❌ NG |
|------|------|------|
| 정상 완료 | "저장했어요" | "저장됨", "저장됐습니다", "저장이 완료되었습니다" |
| 안내 | "잠시 후 다시 시도해 주세요" | "잠시 후 다시 시도하십시오" |
| 확인 요청 | "삭제할까요?" | "삭제하시겠습니까?", "삭제하겠습니까?" |
| 에러 | "응답이 늦어요. 잠시 후 다시 시도해 주세요" | "에러가 발생하였습니다", "오류 발생" |
| 빈 상태 | "아직 등록된 매물이 없어요" | "데이터가 없습니다", "결과 없음" |
| 로딩 | "불러오는 중이에요..." | "Loading...", "처리 중" |

## 3. 숫자·단위 표기

| 영역 | 형식 | 예시 |
|------|------|------|
| 금액 | 억/만원 단위 한국식 | "8억 5,000만원" |
| 면적 (주) | 평 | "120평" |
| 면적 (보조) | ㎡ | "(396.7㎡)" |
| 날짜 | 한국식 | "2026년 4월 22일" |
| 시각 | 12시간제 | "오후 10시 30분" |
| 거리 | km/m | "1.2km", "350m" |

원문 데이터는 SI 단위 (㎡, m), UI에서만 한국식 변환.

## 5. 버튼 텍스트

- 동사 한 단어 또는 짧은 구문
- ✅ "저장", "삭제", "확인", "취소", "다시 시도", "더 보기"
- ❌ "저장하기", "확인합니다", "확인 버튼"

## 6. 에러 메시지 패턴

**원인 + 대응 안내**:

| 패턴 | 예시 |
|------|------|
| 일시 장애 | "응답이 늦어요. 잠시 후 다시 시도해 주세요" |
| 권한 부족 | "이 작업은 중개사 인증이 필요해요" |
| 입력 오류 | "PNU는 19자리 숫자여야 해요" |
| 인증 만료 | "다시 로그인해 주세요" |
| 네트워크 | "인터넷 연결을 확인해 주세요" |

❌ "에러가 발생했습니다" — 원인·대응 둘 다 없음.

## 7. PII 마스킹 (사용자 노출)

| 영역 | 마스킹 |
|------|--------|
| 이메일 | `j***@a***.com` |
| 전화 | `010-****-1234` |
| 사업자번호 | `123-**-12345` |
| 이름 | `김*수` (성 + 별 + 끝자) |
| 주소 | 시군구까지만 ("서울 강남구") |

자동 미들웨어가 응답 직전 마스킹 (`crates/pii/`).

## 8. 법령 인용 (옵션 A 범위 내)

LLM이 *생성*하지 않음. 데이터베이스에 저장된 법제처 원문만 표시.

✅ 정식 명칭 + 조·항·호:
- "국토의 계획 및 이용에 관한 법률 제76조 제5항"

❌:
- "국토계획법 76조 5항" (축약)
- "관련 법령에 따라" (추상적)
