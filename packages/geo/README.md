# @gongzzang/geo

좌표계 변환, 지오메트리 유틸, 공간 연산 헬퍼.

## 책임

- SRID 변환 (4326 ↔ 5179 ↔ 5186 ↔ 3857)
- 주소 → 표준 PNU 정규화
- 거리·면적 계산 (지구 곡면 고려)
- 폴리곤 간소화 (타일용)
- 좌표 검증 (한국 영역 내 여부)

## 의존

- `proj4` (좌표 변환)
- `@turf/turf` (선택, 클라이언트 사이드 보조용)

## 정책

- DB 저장: 항상 EPSG:4326
- 공간 연산: 5179 (UTM-K) 또는 5186 (중부원점)
- 타일 렌더: 3857 (Web Mercator)
- 입출력 시 SRID 명시 강제 (런타임 검증)
